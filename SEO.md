# Google Search Console — SEO Fixes (v2.6)

## Problem
- HTTP and HTTPS both return `200 OK` — duplicate content for Google
- Google crawled `http://iciafrance.com/fr/` and `http://iciafrance.com/en/`

## Fix: `.htaccess` HTTP → HTTPS redirect

```apache
RewriteEngine On
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}/$1 [R=301,L]
```

### Result
```
http://iciafrance.com/fr/  → 301 → https://iciafrance.com/fr/  → 200
http://iciafrance.com/en/  → 301 → https://iciafrance.com/en/  → 200
http://iciafrance.com/     → 301 → https://iciafrance.com/     → 200
```

## Already correct ✓
- **Hreflang** — `<link rel="alternate" hrefLang="fr-FR|en-US|x-default">` generated by Next.js metadata API
- **Canonicals** — self-referencing per language, HTTPS URLs + trailing slash
- **Sitemap** — all URLs use `https://www.iciafrance.com/`, proper hreflang annotations
- **robots.txt** — no blocking of `/fr/` or `/en/`

## Verification
```bash
curl -I http://iciafrance.com/fr/    # should return 301

```
